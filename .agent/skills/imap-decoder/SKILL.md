# imap-decoder Skill

## 概要
IMAP等で取得したメール本文が Quoted-Printable (QP) 形式でエンコードされている場合、日本語が文字化けしたり、正規表現による抽出に失敗する問題を防ぐための専門知識。

## 対応の手順
1. **QPデコード**: 
   メール本文から `=XX` 形式の QP エンコード文字列をバイト列として抽出する。
2. **UTF-8変換**: 
   抽出したバイト列を `Buffer` を介して UTF-8 として解釈する。
   - 正しい変換例: `Buffer.from(qpString, 'latin1').toString('utf8')`
3. **日本語アンカーによる抽出**: 
   デコード後の「日本語キーワード」をアンカーにして、その直後にある確認コード等の必要な情報を抽出する。

## 注意点
- バイト列を介さず `String.fromCharCode` 等で 1 バイトずつ処理すると、多バイト文字である日本語の境界が壊れるため厳禁。
- 対象が UTF-8 のパートであることを前提にする。
